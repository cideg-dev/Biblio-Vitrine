Générateur d'Empreinte (Hash) SHA-256

Comment utiliser cet outil :

  1. Entrez le mot de passe que vous souhaitez utiliser pour la section admin.
  2. Cliquez sur "Générer l'empreinte".
  3. Copiez l'empreinte générée.
  4. Collez-la dans le fichier assets/config/auth.json comme valeur de la clé "hash".

Important : Mémorisez bien votre mot de passe. Si vous perdez votre machine, vous en aurez besoin pour regénérer ce hash.
